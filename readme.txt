Criação de um programa básico para avaliação de filmes. Back end (Java e Springboot) e Front end (Typescript, React, HTML, bootstrap5 e CSS)

Iniciamos a ideia que nos pareceu aceitável e atende aos requisitos para conclusão da disciplina de Orientação à Objetos pela Universidade Federal da Bahia, ministrada pelo Professor Rodrigo Rocha.

Para utilização do programa e compilação completa, após baixar o repositório, utilize o git dentro da pasta front end, com comando: " yarn start "
